Premier Projet

Application Météo

Contexte

Ce projet devait se faire en groupe de trois personnes pour permettre de gagne de l'experience dans le travail de groupe. Ce projet consiste a créer une application permettant d'avoir accès à la météo d'une ville en rentrant le nom de la ville.

Description

Pour ce projet nous avons du utiliser React. Cette application n'a pas de login et est donc accessible par tous. Cette application est composé de deux page standard et une page stack. Les deux pages standard permettent de décrire l'application et de faire une recherche sur une ville. La page stack permet d'afficher le resutat de la météo.
