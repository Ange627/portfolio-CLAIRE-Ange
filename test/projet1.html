Premier Projet

Cas GSB application frais avec le modèle mvc

Contexte

Ce projet devait se faire en groupe de trois personnes pour permettre de gagne de l'experience dans le travail de groupe. Ce projet consiste a créer une application permettant aux employé de rentrer leurs frais lors de déplacement professionnel. Le comptable a accès a ses frais et peut les valider ou les modifier dans le cas où les frais sont mauvaises.

Description

Ce projet a du se faire grâce aux langages HTML, CSS, PHP avec le modèle MVC. Il a fallut créer des fonction permettant d'entrer les données des visiteurs dans la base de données mais aussi de pouvoir les extraire de la base de donnée pour que le comptable puisse les voir. Le comptable pouvait changer l'état des frais des visituer aux états réception des pièces, validation de la demande de remboursement, mise en paiement, remboursement effectué.
